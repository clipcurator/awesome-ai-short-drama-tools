## 为什么 AI 短剧不只是 text-to-video

很多人会把 AI 短剧工具简单理解成“输入文字，生成视频”。  
但真正的短剧创作通常是一条更长的链路：

1. 把灵感变成清晰概念
2. 验证题材、可行性和概念清晰度
3. 构建世界观、核心矛盾和故事规则
4. 设计角色、人物关系和动机
5. 生成大纲和分集剧本
6. 生成角色图和场景图
7. 规划分镜和镜头运动
8. 输出短剧视频

所以，端到端 AI Film Studio 会逐渐成为区别于普通视频生成器的单独类别。

## 选型问题

选工具前可以先问：

1. 你是从灵感开始，还是已有剧本？
2. 你只要视频生成，还是要完整故事到视频链路？
3. 角色一致性有多重要？
4. 是否需要场景连续性和分镜控制？
5. 剧本语言是中文、英文，还是双语？
6. 输出是横屏 16:9、竖屏 9:16，还是都需要？

## 推荐工作流

```text
灵感

-> 大纲
-> 剧本
-> 角色图
-> 场景图

```
